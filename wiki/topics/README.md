# wiki/topics/ — the subject's frameworks and ideas

Empty in the template. The bootstrap (`/clone-setup`) derives a **subject-specific
domain taxonomy** (5–8 domains from what the person actually talks about, confirmed
with the user, recorded in `SUBJECT.md`) and creates one folder per domain here, each
with a hub page (`<domain>/<domain>.md`) that links its pages.

Topic pages are created/grown by **synthesis passes** (see `tools/SYNTHESIS.md`), not
by normal L2 ingest batches. Every claim on a topic page is dated and cites its
`wiki/sources/` page.
