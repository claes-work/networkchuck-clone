# 3 HACKING gadgets you have to TRY!!

## Summary

Chuck shows how to turn cheap Raspberry Pi boards — primarily the ~$40 Raspberry Pi Zero W, plus the Raspberry Pi Pico — into three (plus a bonus) hacking gadgets, framed as a Pi Day project. The through-line is that inexpensive, innocuous-looking hardware can morph into keyboards, mice, ethernet adapters and Wi-Fi attackers to trick a computer, so the video doubles as an awareness piece: he repeatedly says he does not condone illegal hacking and frames the fun uses as pranks ("make your mom think she's hacked").

Despite the title's "gadgets," this video does NOT feature the Flipper Zero, O.MG cable, or a dedicated Wi-Fi Pineapple — it is built entirely around Raspberry Pi DIY builds. It does reference the commercial USB Rubber Ducky (Hak5) as the well-known comparison point for the DIY "BadUSB" builds. Oracle sponsors the video (Pi Day virtual event on building containerized apps on Oracle Cloud Infrastructure).

The three projects: (1) **P4wnP1 A.L.O.A.** — turns a Pi Zero W into a "better BadUSB" with a Wi-Fi web GUI for on-the-fly control; (2) **Pwnagotchi** — an AI-driven Wi-Fi handshake-capturing Tamagotchi-style device that gets smarter the longer it runs; (3) **Kali Linux on the Pi Zero** (including PiTail, which uses an Android phone as screen/controller). A bonus fourth, **PoisonTap**, is shown but Chuck reports it no longer works. He also notes the Pi Pico can be made into a BadUSB for ~$14 for a two-pack (covered in a prior video).

## Key claims (dated 2023-03-10)

- The **Raspberry Pi Zero W** (~$40) can be turned into a "better BadUSB." Chuck recommends the **Zero W** over the Zero W 2 for these projects because the wireless is configured differently on the 2 and the 2 is more expensive; get the **W H** variant (pre-soldered header pins) if you can, useful for the later Pwnagotchi screen.
- **Project 1 — P4wnP1 A.L.O.A.** ("A Little Offensive Appliance"): a BadUSB. Plug it into a computer via USB and it does "bad things" — steals stuff, wiggles the mouse, opens a Rick Roll. Its standout feature: once powered, it broadcasts its own Wi-Fi network you connect to (from a phone/laptop within range), then control the target device on the fly via a web GUI. You can also SSH in (`root` / password `toor` — "root" reversed). It runs Kali Linux under the hood.
- **BadUSB principle (defense-relevant):** what makes a BadUSB "bad" is that when connected it tells the computer it is NOT a USB drive but something else — an ethernet adapter, keyboard, or mouse — so the computer trusts it. Chuck likens it to Pokémon's Ditto. The device is seen by the computer as a mouse and keyboard.
- P4wnP1 uses **HID script** (JavaScript-based, described as a standard) rather than Hak5's proprietary **Ducky Script**, but it can **import Ducky Script** payloads and translate them, giving access to the large community payload library. Possible payloads listed: capturing logins, disabling security, grabbing passwords, Rick Roll, PowerShell reverse shell, exfiltration. Setup uses the Raspberry Pi Imager to flash the A.L.O.A. image (latest release a 2020 beta) to a 32 GB micro SD card.
- The **USB Rubber Ducky** (from Hak5) is the most popular commercial BadUSB; it "looks like nothing" by design. Chuck notes he ordered plain USBs off Amazon that look identical — the social-engineering idea is to trick someone into plugging it in.
- The **Raspberry Pi Pico** can also be made into a BadUSB — ~$14 for a two-pack — using Ducky Script payloads (covered in a prior Chuck video).
- **Project 2 — Pwnagotchi:** a Tamagotchi-style device that uses **AI to tune its parameters** to get better at capturing Wi-Fi handshakes for whatever environment it is exposed to. The longer you leave it running and the more wireless traffic it sees, the smarter it gets. It can deauth ("do some deauth") and pull handshakes. Chuck plans to carry a USB power bank so he can run it in permitted environments. Setup requires editing/creating a `config.toml` file in the boot drive to enable the web UI and set admin credentials; the Pi connects to the host over USB via RNDIS ethernet gadget, reached at a static IP (`10.0.0.2:8080`). Works on Pi 3/4 too, not just the Zero.
- **Project 3 — Kali Linux on the Pi Zero:** installable directly via the Raspberry Pi Imager ("Other specific purpose OS" → Kali Linux). A variant, **PiTail**, lets you use an Android phone as the screen and controller for the Pi, making a portable pocket hacking environment. (A P4wnP1 image is also offered in the Imager but did not work for Chuck — he recommends installing from GitHub instead.)
- **Bonus — PoisonTap** (by Sammy Kamkar, 2016): promised to sniff all network traffic from a locked/unattended computer and exfiltrate it to your web server. Chuck says it **no longer works** — he tried for hours and got no data. Included as a "cool idea" that has aged out.
- **Awareness/defense framing:** Chuck repeatedly states he does not condone anything illegal; the intended use is harmless pranks on friends/family. The implicit defense lesson is the danger of plugging in unknown USB devices, since a trusted-looking stick can present itself as a keyboard/mouse/ethernet adapter and attack "unlike every computer I know of."
