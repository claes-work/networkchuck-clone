# Should I Learn LINUX with the CCNA | CCNP? - Network Engineer

## Summary
Chuck answers a common viewer question: should you learn Linux alongside the CCNA/CCNP? His short answer is yes — Linux belongs on every network engineer's list — but the timing is flexible. If you're just starting the CCNA, don't stress about it and focus on the core material; once you finish the CCNA and move toward the CCNP, start folding Linux in. He frames Linux (along with Python and networking) as core to the "next generation network engineer" skill set, because so much Cisco gear and so many enterprise servers run on Linux. This clip is also part of his "10 Days of Christmas" giveaway series.

## Key claims
- (2017-12-19) You don't need to master Linux to start; if you're just beginning the CCNA, focus on the core material and don't get bogged down — Linux is "a whole new world" like learning Windows for the first time.
- (2017-12-19) Chuck still recommends adding Linux to your training path because most of Cisco's newer products — security appliances, Firepower, Collaboration/Call Manager — are Linux-based, and back-end administration sometimes requires Linux work.
- (2017-12-19) The majority of the world's web servers run on Linux (he cites Tomcat/Red Hat), and many companies and appliances are Linux-only.
- (2017-12-19) Cisco Nexus data-center-class switches use a Linux-based file system.
- (2017-12-19) Chuck groups Linux with Python and networking as the skill set of the "next generation network engineer."
- (2017-12-19) Chuck describes himself as no Linux expert but says he uses it: Debian, Python inside Linux, OpenVPN on a Debian box, Raspberry Pi (Linux-based) — and knowing "a little bit" has helped his career, especially with Collaboration (every Collaboration app runs on Red Hat Linux).
- (2017-12-19) To learn Linux he suggests cheap Udemy courses or CBT Nuggets (which he says has amazing Linux courses).
- (2017-12-19) Getting started is easy: download VirtualBox, grab an Ubuntu ISO, install it, and play — Ubuntu has a GUI plus command line, and it's great for learning Python.
- (2017-12-19) Practical advice for post-CCNA / pre-CCNP students: devote maybe a day a week to Linux.
- (2017-12-19) He references David Bombal's "Python for Network Engineers" course, noting it exposes you to Linux because the Python scripts are run from a Linux box.
