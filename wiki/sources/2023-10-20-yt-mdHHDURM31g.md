# Windows has a SECRET MODE??

## Summary

A Windows power-user tip short demonstrating "God Mode." On a Windows laptop, create a new desktop folder and rename it to a specific string; on hitting enter, the folder transforms into "God Mode" — a single window listing all of Windows' scattered Control Panel settings. The presenter's favorite part: any of these settings can be right-clicked and turned into a desktop shortcut (e.g. right-clicking "Sound" to create a shortcut straight to the volume mixer).
