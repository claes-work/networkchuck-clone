# GOODBYE Microsoft certifications!! (killing off the MCSA, MCSE, MCSD)

## Summary

A February 2020 cert-news reaction video. Chuck reports that Microsoft is
retiring nearly all of its technology-based certifications — the MCSA, MCSD,
and MCSE families — effective **June 30, 2020**, and walks through the official
Microsoft blog post to show what is going away and what replaces it. He frames
it as a "Microsoft cert-pocalypse" and compares it directly to Cisco's then-recent
certification overhaul that killed off associate-level tracks. The big takeaway:
Microsoft is not replacing the old technology-specific certs with new versions
(no Windows Server 2019 or SQL Server 2019 certification exists) — it is pivoting
entirely to **role-based, Azure/cloud-focused certifications**. Chuck ends
somewhat conflicted: he agrees the cloud is the future and thinks the move is a
good step, but he is confused about how on-prem administrators will now prove
their skills to employers when all the certs are cloud-based. He discloses he
authored the AZ-900 and AZ-103 (Azure) training on CBT Nuggets and holds both
certs.

## Key claims (dated — the change + his take)

- **[2020-02-27] The change:** Starting **June 30, 2020**, Microsoft retires the
  MCSA, MCSD, and MCSE certifications — you can no longer test on them after that
  date (paraphrase).
- **[2020-02-27] Grandfathering:** If you already hold (or earn before the
  deadline) one of these certs, it stays a valid/active certification for **two
  years** — i.e., good until **June 30, 2022** — after which it moves to a
  "retired" state (paraphrase).
- **[2020-02-27] What's retiring:** The big flagship tracks are going — Windows
  Server 2012/2016 MCSAs, MCSE Core Infrastructure, etc. — along with all their
  underlying exams (paraphrase).
- **[2020-02-27] No like-for-like replacement:** Traditional technology-based
  certifications (e.g., "get certified in Server 2016") are going away and are
  NOT being replaced with equivalents; per Microsoft there will be **no Windows
  Server 2019 or SQL Server 2019 certification** (paraphrase).
- **[2020-02-27] The replacement model:** Microsoft is going all-in on
  **role-based certifications**. Server 2019 / SQL Server 2019 content will be
  folded into role-based certs "on an as-needed basis" within the Azure
  Infrastructure, Apps & Infrastructure, and Data & AI solution areas (paraphrase).
- **[2020-02-27] Cloud pivot:** The recommended role-based paths are dominated by
  Azure and Microsoft 365 cloud certs (e.g., Azure Administrator path: AZ-900
  fundamentals → AZ-103 → AZ-400). Chuck reads this as "Azure, the cloud, is
  replacing everything" (paraphrase).
- **[2020-02-27] Cisco parallel:** He explicitly compares this to Cisco's recent
  certification overhaul — Microsoft is "basically doing exactly what Cisco did"
  by killing off associate-level certs (the MCSA being roughly the Microsoft
  equivalent of the CCNA) (paraphrase).
- **[2020-02-27] His take — mixed:** He thinks the cloud shift is a good step
  since the industry is genuinely moving to cloud, but he is confused/concerned
  about how engineers will demonstrate **on-prem** server skills to employers
  when every cert is now cloud-based; he speculates Microsoft is deliberately
  pushing certified engineers toward the cloud to drive cloud adoption (paraphrase).
- **[2020-02-27] Hope for the industry:** He hopes HR departments and hiring
  managers will move with Microsoft and start recognizing/seeking the new
  role-based certs quickly (paraphrase).
- **[2020-02-27] Disclosure:** Chuck authored the **AZ-900** and **AZ-103** Azure
  training courses on CBT Nuggets and currently holds both certifications
  (self-reported).
