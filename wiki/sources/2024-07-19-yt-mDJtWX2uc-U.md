# Helpdesk IT Challenge!! How many can you get right?

## Summary

A rapid-fire CompTIA A+ quiz short using Boson practice-exam questions, with new
video editor **Mike** answering (and mostly getting them wrong). Questions cover
sandboxing vs. honeypots, impact printers, DVI cabling, and projector overheat
shutoff. Doubles as a soft plug for Boson's A+ practice exams. Light
certification-prep content; the takeaway is that structured practice exams make
IT learnable.

## Key points

- 2024-07-19 — A sandbox (not a honeypot) is used to isolate an application
  within a controlled environment.
- 2024-07-19 — Lasers and inkjet are NOT impact printers (Mike answered these and
  was marked incorrect).
- 2024-07-19 — A DVI cable connects a display, not a Bluetooth speaker (Mike's
  answer was wrong).
- 2024-07-19 — An overheating bulb can cause a projector to automatically shut off
  (marked correct).
- 2024-07-19 — Soft plug for Boson A+ practice exams as a learning tool.
