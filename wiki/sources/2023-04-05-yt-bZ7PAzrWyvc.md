# "Kids vs. MALWARE!!" (2023-04-05)

## Summary

A demo of a **disposable / cloud browser** as malware protection. Chuck runs an experiment
where he has his two daughters click sketchy links and download malware ("SpongeBob Fun
Pack"): one daughter uses the NetworkChuck cloud browser, the other a normal browser like
most viewers use. The one on the disposable browser stays protected; the one on the normal
browser gets hit. The mechanism: the disposable browser spins up a brand-new computer
somewhere else in the world, you access the risky website through that remote machine, and
when done you delete both the browser and the computer it ran on — so malware never
touches your real device. (Family framing is self-reported; the daughters are not named.)

## Key points

- 2023-04-05 — A disposable/cloud browser protects against malware by running the browsing
  session on a brand-new remote computer, not your own device.
- 2023-04-05 — After use, you delete the browser and the underlying remote computer, so any
  downloaded malware is destroyed with it.
- 2023-04-05 — Demo contrast: disposable-browser user stayed safe; normal-browser user was
  infected by the same sketchy download.
- 2023-04-05 — As long as you browse risky links through the disposable cloud browser,
  malware "doesn't matter." <!-- ★ L3-candidate: recurring NetworkChuck security stance — disposable/cloud browser as the default way to open untrusted links; appears across multiple shorts -->
