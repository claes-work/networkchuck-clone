# What does a PROXY actually do??

## Summary

A short explainer of what a proxy does. When you connect to a website you send a message that includes your IP address, which can reveal a lot about you. A proxy is a server that sits between you and the website: it intercepts your message and the website sees the proxy's information instead of yours. Chuck uses the example that a proxy in Ireland makes "the whole internet think you're Irish." Ends by pointing to a full video on building your own proxy.

## Key points

- (2023-11-27) Connecting to a website sends a message that includes your IP address.
- (2023-11-27) An IP address can tell companies a lot about you, which is the reason to use a proxy.
- (2023-11-27) A proxy is a server that stands between you and the website; it intercepts the request so the website sees the proxy's information instead of yours.
- (2023-11-27) Example: a proxy located in Ireland makes the whole internet think you're Irish (location masking).
