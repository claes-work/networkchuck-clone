# Unboxing $48,000 worth of Mac Studios

## Summary

A short, casual "chill unboxing" second-channel video in which Chuck Keith unboxes four
brand-new top-of-the-line Mac Studios that Apple sent him on loan (not to keep) to build
what he calls the most powerful local-AI cluster you can build with consumer hardware.
The video is deliberately low-key ASMR-style — Chuck is recovering from the flu — and
focuses on the hardware, the packaging, and the cluster's intended purpose. It pairs with
the main-channel Mac Studio cluster / "Ethernet is DEAD" build (2025-12-20). Each machine
is numbered as part of the cluster (Mac Studio 3 and 4, plus the four total unboxed), and
Apple also shipped Thunderbolt 5 cables to interconnect them.

## Key claims (dated — hardware + purpose)

All claims dated 2026-01-27 (source date):

- Chuck unboxes **four brand-new Mac Studios**, sent to him by Apple **on loan** (to be
  returned, not kept) — return shipping labels included in the boxes.
- Each unit is a **top-of-the-line Mac Studio**: **32-core CPU, 80-core GPU, 512 GB of
  unified RAM, 8 TB of storage** — described as the best Mac configuration you can get.
- Chuck emphasizes the 512 GB of RAM as remarkable ("that's a hard drive, but it's RAM").
- Apple also shipped **Thunderbolt 5 cables** to interconnect the machines for the cluster.
- Purpose: build **a local AI cluster** — Chuck claims it may be **the most powerful local
  AI cluster** you can find / that there has ever been, and the most powerful you can build
  with **consumer hardware**.
- He estimates each machine's value at roughly **$12,000** ("holding like $12,000 in my
  hand"); the four together frame the ~$48,000 title figure.
- Contrast with his previous cluster (main-channel 2025-12-20 build): the last one used
  **five Mac Studios** with only ~**64 GB of RAM each** (maybe less), so a single one of
  these new units has more RAM than that entire previous cluster combined.
- He hopes the **bandwidth and latency between machines** running AI models will be much
  better than the previous cluster (not yet tested at time of recording).
- Practical advantage claimed: the cluster **draws little power, outputs little heat, and
  is quiet** — so it does not need a data center or cooled server room and can be placed
  anywhere.
