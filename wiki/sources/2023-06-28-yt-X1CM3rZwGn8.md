# you NEED to learn Windows RIGHT NOW!!

## Summary

An entry in Chuck's signature **"you need to learn X RIGHT NOW"** format, applied to
Microsoft Windows from an IT-professional / sysadmin perspective (not the end-user
perspective). Chuck's thesis: Windows holds ~75% of the desktop market, so most
companies run it, which means IT jobs — especially entry-level help desk — require
Windows *admin/support* knowledge, not just user familiarity. He also frames Windows as
essential for aspiring ethical hackers because it is usually the target being attacked
(he cites pursuing Hack The Box's CPTS cert, which lists Windows as a prerequisite).

The video is structured as: (1) why learn Windows (jobs + hacking); (2) a sponsor coffee
break where he addresses the "Windows is insecure" objection — arguing Windows gets
hacked because it's *popular*, not inherently insecure (NordVPN sponsor read covering
Threat Protection and obfuscated servers); (3) "Windows history as fast as possible" — a
60-second speed-run from Windows 1.0 (1985) through Windows 11, run in VirtualBox VMs;
(4) the hidden world of **Windows Server** and what a server actually does; (5) a deep
demo of **Active Directory**, **network shares (SMB)**, and **Group Policy**; (6) an
intro to **Microsoft Azure** / Azure AD and cloud VMs; (7) two ways to build a home
Windows lab — Azure (free tier, $200 credit, pay-as-you-go) and local VirtualBox/Parallels
VMs; and (8) certifications and courses to learn Windows in a structured way. He plugs
his own in-development Windows Fundamentals course on NetworkChuck Academy.

The teaching core is the user-vs-admin knowledge distinction (admin knowledge is "the
knowledge you'll get paid for"), and the demonstration that a Windows Server centrally
manages an entire company: Active Directory handles authentication so a user can log into
any company machine, File & Storage Services provides network shares over SMB, and Group
Policy controls everything about the workstations — even down to blocking Command Prompt
or denying USB drives (he ties USB blocking to defending against a Flipper Zero BadUSB
attack). He notes local Group Policy exists on every Windows machine and invites viewers
to try editing it, and mentions he runs a Windows Server with Group Policy at home to
control his daughters' computers.

## Key claims (dated — the concepts)

- **[2023-06-28]** Windows holds over 75% of the desktop OS market, so most companies run
  it — which means most companies need people with Windows skills. (paraphrase)
- **[2023-06-28]** For getting a first IT job (usually help desk), Windows admin/support
  knowledge is effectively required — knowing how to support, administer, fix and break
  it, not just having used it. (paraphrase)
- **[2023-06-28]** Aspiring ethical hackers must learn Windows because it's usually the
  target of the attack — "you can't hack it if you don't know it." Chuck notes Windows is
  a prerequisite for Hack The Box's CPTS certification. (paraphrase)
- **[2023-06-28]** The core distinction the video teaches: **user knowledge vs. admin
  knowledge** — admin knowledge is the knowledge companies pay for. (paraphrase)
- **[2023-06-28]** Windows is not inherently more insecure than other OSes; Windows users
  get hacked because Windows is *popular* — malware authors write for the masses / the
  biggest target. (paraphrase, framed within the NordVPN sponsor read)
- **[2023-06-28]** Windows history highlights: Windows born 1985 (GUI for MS-DOS);
  Windows 3.0 (1990); Windows 95 (taskbar + Start button); Windows 2000 introduced
  **Active Directory**, "which we still use today"; Windows XP (2001, his favorite,
  brought Remote Desktop); Vista introduced Windows Search, Windows Defender and User
  Account Control; Windows 7 (2009); Windows 10 (2015); Windows 11 (current). (paraphrase)
- **[2023-06-28]** Behind the consumer editions sits a parallel line of **Windows Server**
  releases (Windows NT 3.1 in 1993 up to Windows Server 2022) that most users don't know
  exist. A server "provides services"; in a company you might run Windows Server 2022 in
  the data center serving Windows 11 desktops to employees. (paraphrase)
- **[2023-06-28]** **Active Directory** centrally manages user accounts (via "Active
  Directory Users and Computers") so an admin creates a user once and that user can log
  into any company machine — the AD server handles authentication — instead of creating
  accounts on each machine individually. This is a must-learn skill and a common help-desk
  task. (paraphrase)
- **[2023-06-28]** A Windows Server also provides **centralized storage** — network shares
  created via the File and Storage Services role, accessible to permitted users over the
  **SMB** protocol. (paraphrase)
- **[2023-06-28]** **Group Policy** lets a server control everything about all company
  workstations, down to the smallest setting, applied to computers or users — "insanely
  powerful" but also dangerous, so be careful. Every Windows machine also has **local
  group policy** viewers can edit (e.g. disabling Command Prompt, or denying read access
  to removable USB drives to defend against a Flipper Zero BadUSB attack). (paraphrase)
- **[2023-06-28]** **Microsoft Azure** ("the cloud is just someone else's computer") can
  do everything on-prem servers do at scale — Azure Active Directory, cloud Windows VMs,
  even remote desktops running in Azure. Cloud/Azure skills are "big money." (paraphrase)
- **[2023-06-28]** Lab option 1 — **Azure free tier**: requires a credit card (identity
  verification only; no charge without consent), gives a $200 credit for 30 days, and is
  pay-as-you-go afterward (~12¢/hour for a Windows 11 VM, billed only while running). A
  running server + Windows 11 VM cost him $3.64 over a few days. He demos creating a VM,
  connecting via RDP, setting a budget/cost alert, and deleting the resource group to
  clean up. (paraphrase)
- **[2023-06-28]** Lab option 2 — **local VMs**: check Task Manager for spare RAM/disk
  first (he has 64 GB; 4 GB laptops shouldn't bother, 8–16 GB can run a few). Download the
  free **VirtualBox** hypervisor plus a free Windows 11 or Windows Server 2022 trial ISO;
  Windows 11 needs 4 GB RAM minimum. Apple Silicon (M1/M2) Macs need an ARM OS — easiest
  via **Parallels** (paid, 14-day trial). (paraphrase)
- **[2023-06-28]** Cert path: **CompTIA A+** is his recommended starting point (broad,
  marketable, was his own first cert, but only lightly covers Windows). Microsoft's own
  certs teach a lot but change too often to be reliably marketable; he flags **MD-102**
  (Modern Desktop Administrator Associate), the **Windows Server Hybrid Administrator
  Associate** (AZ-800/AZ-801), and the Azure track **AZ-900** (Fundamentals) → **AZ-104**
  (Administrator Associate) as strong, with Windows knowledge as a prerequisite for the
  cloud-focused ones. (paraphrase)
- **[2023-06-28]** He's building a **Windows Fundamentals course** on NetworkChuck Academy
  ("enough to be dangerous" and a foundation for Azure or hacking), and also recommends
  Hack The Box Academy's text-based Windows Fundamentals course with built-in labs.
  (paraphrase)
