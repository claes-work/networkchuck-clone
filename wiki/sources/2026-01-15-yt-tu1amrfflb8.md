# NEW Raspberry Pi AI Hat Is a Massive Upgrade

## Summary

Chuck unboxes and installs the **Raspberry Pi AI Hat Plus 2**, contrasting it with the
original AI Hat Plus. The original had zero RAM (it had to steal RAM from the Pi) and
was limited to camera-based AI like object detection and image recognition. The new
Hat 2 has **8 GB of onboard RAM**, ships with a required heat sink (more power = more
heat), and lets you run larger AI models **100% locally**. He installs it on a
Raspberry Pi 5 (16 GB) via standoffs and a GPIO riser, flashes a fresh Raspberry Pi OS
image, installs the Hailo software and Docker, and starts the Hailo AI service so the
Pi treats the Hat as a **local AI accelerator**. A large language model is pulled down
directly onto the Hat — no cloud, no API keys — and can be prompted from the terminal
or via **Open Web UI** in another container for a ChatGPT-style web interface.

## Key points

- 2026-01-15: LLM pulled directly onto the Hat — "No cloud, no API keys, it's all
