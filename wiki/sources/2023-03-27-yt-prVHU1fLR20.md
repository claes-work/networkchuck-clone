# 30 Windows Commands you CAN'T live without

> Windows-specific companion to Chuck's "60 Linux Commands" video. Domain-tagged
> `linux-terminal` as the closest command-line/sysadmin fit in the taxonomy — there is
> no dedicated Windows domain. All examples are CMD / PowerShell / Windows Terminal.

## Summary

A rapid-fire walkthrough of essential Windows command-line tools, framed as the Windows
counterpart to Chuck's popular "60 Linux Commands" video (the on-screen count runs to
~35 despite the title). Chuck runs everything from the Command Prompt "beast mode"
(elevated / run-as-administrator) and organizes the tour into loose clusters: privacy
and file-hiding tricks (`copy /b`, `cipher`, `attrib`), Wi-Fi credential recovery
(`netsh wlan show profile`), system info and remote copy (`systeminfo`, `scp`),
launcher/navigation shortcuts (`cmd` from Explorer, `explorer .`, `subst`), cosmetic
customization (`color`, `prompt`, `title`), a large `curl` section (weather, link
unshortening, site status, public IP, QR codes, latest YouTube/Twitter post, dictionary,
and even ChatGPT from the CLI), miscellaneous fun (`start`, temp-file cleanup, `telnet
telehack.com` for Star Wars/aquarium), the F1–F9 command-history function keys plus
`doskey /history`, and a closing pitch to graduate from Command Prompt to Windows
Terminal and PowerShell. Sponsored by ITProTV (ACI Learning).

## Key claims (dated 2023-03-27; paraphrase unless quoted)

**Elevated shell setup**
- You can make Command Prompt always launch as administrator ("beast mode") by editing
  the shortcut: Properties → Advanced → "Run as administrator" by default.

**Privacy / hiding files**
- `copy /b image.png + secret.zip newphoto.png` hides a zip archive inside an image
  file — to anyone else it looks like an ordinary photo.
- `cipher /e <folder>` encrypts every file in a folder so only the owning account can
  open them.
- `attrib +H +S +R <folder>` hides a folder (system + read-only + hidden attributes);
  reverse with the minus signs (`attrib -H -S -R`) to bring it back.

**Wi-Fi credentials**
- `netsh wlan show profile` lists every Wi-Fi network the machine has connected to.
- `netsh wlan show profile <SSID> key=clear` reveals that network's saved password in
  plain text; a one-liner (loop) can dump all of them, and `>> wifi-passwords.txt`
  writes them to a file. Save the one-liner as a `.bat` file (e.g. `wifi.bat`) for reuse.

**System info & remote copy**
- `systeminfo` prints a full system information report.
- `scp` (secure copy) copies a file to a remote server over SSH:
  `scp <file> user@server:/path` — Chuck says he uses this all the time.

**Navigation / launcher shortcuts**
- Type `cmd` in the Explorer address bar to open a Command Prompt at that folder; type
  `explorer .` in the terminal to open an Explorer window at the current directory.
- `subst <drive>: "<path>"` maps any folder to a virtual drive letter; `subst <drive>: /d`
  removes the mapping.

**Cosmetic customization**
- `color <bg><fg>` sets background/text colors (first digit = background, second = text;
  `color /?` lists codes; green-on-black is `color 02`).
- `prompt` changes the prompt text (e.g. `prompt [Hey beautiful]$G`); `prompt` alone
  resets it. `title <text>` renames the terminal window.

**curl (built into Windows 10/11)**
- `curl wttr.in/<city>` — weather in the terminal.
- `curl -sIL <short-url> | findstr Location` — unshorten a link / see where it redirects.
- `curl -Is <url>` — check whether a website is up (looks for HTTP `200`).
- `curl checkip.amazonaws.com` — show your public IP address.
- `curl qrenco.de/<url>` — render a QR code as ASCII in the terminal.
- `curl` against specific endpoints can fetch a YouTuber's latest video title or a
  user's latest tweet, define a word (`curl dict.org/d:<word>`), and even query ChatGPT
  from the command line.

**Miscellaneous**
- `start <website>` opens a URL in the default browser from the terminal.
- Temp-file cleanup commands free disk space (delete temp files, then a restore command).
- `telnet` (enable via Windows Features) → `telnet telehack.com` gives retro fun like
  ASCII Star Wars and an aquarium; exit with Ctrl+C.

**Function keys & history**
- F1–F9 manipulate command history: F1 retypes the previous command char-by-char, F3
  copies the entire previous line, F5 cycles through previous commands, F7 opens a
  selectable list of previous commands (Chuck calls this the coolest), F8 recalls the
  first command, F9 recalls a command by its number. Chuck admits several (F2, F4, F9)
  have unclear use cases.
- `doskey /history` prints the full list of commands entered in the session.

**Graduate to Terminal / PowerShell**
- Windows Terminal (built into Windows 11, installable on 10) is the modern replacement
  for Command Prompt; killer feature is tabs. Shift+Right-click a folder → "Open in
  Terminal". It also lets you drag-and-drop a file into an `scp` command to fill the path.
- Chuck's closing advice: use PowerShell — "PowerShell is better than Windows Command
  Prompt in a lot of ways"; it can switch between Command Prompt, PowerShell, and Azure
  Cloud Shell inside one Terminal.
