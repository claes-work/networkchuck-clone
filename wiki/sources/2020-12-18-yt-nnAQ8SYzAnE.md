# hack like Mr. Robot

## Summary

A short pop-culture hacking-education video (Chuck's "day three" giveaway) built around the physical hacking gadgets sold by **Hak5** — the same devices that show up in TV shows like *Mr. Robot* (the USB Rubber Ducky) and *Silicon Valley* (the Wi-Fi Pineapple). Chuck's angle is that these are not TV props: they are legit tools that real professional penetration testers use, and he showcases them one by one while explaining what each does. The video doubles as a promotion/giveaway: Hak5's Darren provides ten store gift cards, and viewers enter a contest with a secret passphrase ("rubber ducky") revealed at the end.

The gadgets he walks through: the **Wi-Fi Pineapple** (~$100, for wireless pen-testing / hacking into enterprise Wi-Fi), the **Key Croc** (a USB keylogger that captures keystrokes remotely, can impersonate other USB devices like storage or Ethernet, and runs payloads triggered by typed words), the **Shark Jack** (a USB-shaped device with Ethernet that auto-scans a network and runs Nmap/scripts when plugged into an Ethernet port), the **USB Rubber Ducky** (the classic keystroke-injection tool famous for cracking a Windows password in ~15 seconds), and the **Screen Crab** (a man-in-the-middle HDMI device placed between a computer and its monitor to remotely capture screenshots).

The framing is explicitly ethical: Chuck repeatedly stresses these are professional tools to be used only with permission, and that the appeal is that "hacking gadgets are cool" plus the promise of future hands-on videos demonstrating each one. A mid-video "flash giveaway" also promotes David Bombal's *Python for Network Engineers* course (first 500 clicks). Store: shop.hak5.org.

## Key claims (dated — what he shows + ethics)

All 2020-12-18 (paraphrase unless quoted):

- **Ethics first**: these are legit tools that real professional hackers use ethically; never hack anyone without permission — "please don't get in trouble."
- **Hak5** is a company that makes gadgets and tools for hackers; several have appeared on TV (USB Rubber Ducky on *Mr. Robot*, Wi-Fi Pineapple on *Silicon Valley*).
- **Wi-Fi Pineapple**: professional pen testers use it to hack wireless / enterprise networks; it does "pretty much anything you want to hack with Wi-Fi" and costs about $100.
- **Key Croc**: a small USB keylogger you plug into a computer to see what people type remotely and in real time; it can pretend to be other devices (storage, Ethernet) to trick the computer, and run payloads / hacking scripts triggered by typed words (e.g. capture a password when the user hits Ctrl+Alt+Delete) — all done remotely.
- **Shark Jack**: a USB-shaped device with Ethernet on the end; plug it into an Ethernet port and it auto-hacks — scans the network, runs Nmap, executes scripts; a green light signals when it's done.
- **USB Rubber Ducky**: a long-standing go-to pen-tester tool whose claim to fame is cracking a Windows password in ~15 seconds when plugged in.
- **Screen Crab**: performs a man-in-the-middle attack between a computer and its monitor (HDMI in one end, out the other) and remotely captures/sends screen images.

- **Giveaway**: Hak5's Darren provided ten gift cards to the Hak5 store; enter via a linked contest using the passphrase "rubber ducky," within 24 hours.
- **Flash giveaway (mid-video)**: David Bombal is giving away 500 copies of his *Python for Network Engineers* course to the first 500 people to click the description link.
