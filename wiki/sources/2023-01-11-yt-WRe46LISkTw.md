# ChatGPT will make you better

## Summary

Chuck's **first major ChatGPT / generative-AI video** (January 2023), marking his pivot
into the gen-AI era. He addresses the fear that ChatGPT will take everyone's jobs and
rejects it: his core stance is that ChatGPT won't replace IT professionals, it will make
them "awesome" — a productivity multiplier and study/coding buddy, "the greatest IT tool
ever in history." He notes ChatGPT is free at the time ("research/open preview"), is the
third iteration of OpenAI's model, and was trained on text from the internet ("we're like
its parents").

He then demos practical IT uses live: (1) **Coding** — generating Python that deploys a
Linode VM via the Linode API using plain-language requests, then refining it conversationally
(swap distro to CentOS, region near Dallas) while ChatGPT keeps context; explaining existing
bash scripts in plain language; debugging by pasting errors back over a long conversation;
and asking it to improve code. (2) **Studying** — "explain it like I'm five," comparing VMs
vs. containers, drilling into container security risks, and generating self-quizzes/multiple-choice
flashcards. (3) **Hacking/cybersecurity** — Nmap commands that evade firewall detection (with
each flag explained), turning that into a Python script, and writing a pen-test report — while
noting ChatGPT refused to write a phishing email because OpenAI tightened its policies after
the December release. (4) **Cloud & networking** — Azure CLI to deploy VMs + Kubernetes, the
same in AWS, and Cisco/Juniper CLI for a VPN tunnel between routers. (5) **Writing** — turning
rough project notes into an entertaining blog post, resume bullet points, and emails, then
re-toning them ("more informal, more fun"). He also shows ChatGPT role-playing as a Linux
machine / Python IDE. Linode is the sponsor. He closes by choosing optimism over pessimism:
you can't control whether AI takes jobs, but you can use it right now to make yourself better.
