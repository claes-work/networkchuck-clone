# Here's why everyone is worried about Google Chrome

## Summary

A short on Google's plan to build integrated proxies into Chrome. Chuck gives a quick refresher (a proxy is a server standing between you and the website so the site doesn't see your private info), then raises the privacy concern: you are now handing your private information to the proxy instead of the website — and if Google's servers were hacked, that data would be at risk. Delivered with sarcasm ("don't worry Google's got everything under control"), it ends by nudging viewers to run their own proxy servers instead of relying on Google.

## Key points

- (2023-11-27) Google wants to add integrated proxies into Chrome.
- (2023-11-27) A proxy is a server that stands between you and the website so the site doesn't see your private information.
- (2023-11-27) Concern: instead of sending your private info to the website, you send it to the proxy — and if Google's servers get hacked, many people's private info could be at risk.
