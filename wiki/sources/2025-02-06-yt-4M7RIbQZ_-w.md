# Deep Research.....but Open Source

## Summary

Chuck walks through an **open-source alternative to OpenAI's "Deep Research"** agent — a
GitHub project called **`deep-research`** created by a developer named David (found via a
Twitter/X post). OpenAI's Deep Research is gated behind a **$200/month ChatGPT Pro**
subscription; the open-source implementation gives roughly the same multi-step research
capability on a **pay-as-you-go** basis using your own API keys (typically "pennies" for
light use).

He explains what a "deep research" agent is: instead of a fast one-shot answer, it spends
5–30 minutes doing multi-step reasoning — going out to web pages, reading them, doing data
analysis, running follow-up searches, and returning a cited, PhD-quality report. Both the
proprietary and open-source versions here run on **OpenAI's o3-mini (high)** model, and
both ask clarifying questions before starting.

Setup (done together, live): clone the GitHub repo, then supply two API keys — a
**Firecrawl API key** (free hosted tier, up to 500 credits, used for web crawling/scraping)
and an **OpenAI API key** (pay-as-you-go, not covered by a ChatGPT subscription). Chuck runs
the Node.js app **inside a Docker container** (his preferred approach for isolation;
on Windows this runs under WSL) rather than bare `npm`. Config lives in a `.env.local` file
plus a `Dockerfile`, `docker-compose.yml`, and a small edit to `package.json`. The tool lets
you set the research **breadth** (how wide) and **depth** (how deep), and saves output to a
`report.md` markdown file.

He runs two side-by-side comparisons against OpenAI Deep Research — "which animal is better,
cats or dogs" (both conclude dogs) and "which OS is best for IT pros: Mac, Linux, or
Windows" (both conclude: no single OS — use all three / hybrid strategy). The open-source CLI
was faster but hit Firecrawl **rate limits** on the free hosted tier, which is why he notes
you'd eventually want to run it locally. A closing note mentions Hugging Face announcing its
own open-source agentic deep-search framework.

Framing throughout: doing it yourself in the command line "just feels cooler," the code is
all open and inspectable (vs. OpenAI keeping things "very mysterious"), and it's far cheaper
than $200/month.
