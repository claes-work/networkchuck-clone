# Learning Python is HARD!! - CCNA | CCNP Network Engineer

## Summary
Chuck argues that Python is worth learning but painful to start, contrasting how naturally CCNA/CCNP material came to him versus how many times he burned out and restarted with Python. He blames dry, theory-first courses and endless "hello world" exercises, and says the breakthrough was learning by doing — specifically applying Python to networking by scripting a real router/switch. The video also promotes and gives away 10 copies of David Bombal's "Python for Network Engineers" Udemy course, framed around a Black Friday sale.

## Key claims
- (2017-11-23) Python is awesome, but learning it is frustrating — CCNA/CCNP came naturally to Chuck while Python felt "like learning Chinese."
- (2017-11-23) Traditional Python courses are too dry and theory-heavy; hitting a wall of programming basics early kills motivation, especially for a network person who wants networking-relevant practice.
- (2017-11-23) He restarted learning Python many times, hitting burnout and quitting, then coming back months later — never getting past the beginning.
- (2017-11-23) Getting started is the hardest part of any journey (networking, CCNA/CCNP, or Python); once you get rolling it becomes rewarding.
- (2017-11-23) The best way to learn to code, per coders' own advice, is to just start coding — for Chuck that meant taking what he'd normally do on the CLI and doing it in a Python script against a router/switch.
- (2017-11-23) Networking and programming worlds are merging; network engineers should start learning a little Python alongside their CCNA because the industry is moving toward network programmability / SDN (automation, deploying many devices at once instead of one at a time via CLI).
- (2017-11-23) Python is not required to be a typical network engineer right now, but viewers who want more money, success, freedom, and to stand out ("be a unicorn") should learn it to get ahead of the curve.
- (2017-11-23) He got up and running in GNS3 with Python and programmed a switch in a single day, and recommends David Bombal's "Python for Network Engineers" Udemy course as what got him started.
- (2017-11-23) GNS3 is like Packet Tracer — a way to emulate/simulate routers and switches (and can use Cisco VIRL images, which are not free).
- (2017-11-23) Holidays are a good time to get solid study in — step away from the hustle but still study.
