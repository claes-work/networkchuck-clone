# Milla Jovovich made an AI memory tool…..it's pretty good

## Summary

Chuck tries **Mem Palace**, an AI memory tool made by actress Milla Jovovich and her friend Ben, billed as the highest-scoring AI memory system ever benchmarked (free, ~5,000 GitHub stars). It gives AI "permanent memory" by mining past AI conversations and organizing them into a memory-palace structure (wings = people/projects, halls = types of memory, rooms = specific ideas). He installs it live (`pip install mem palace`, `mem palace init`), points it at his Obsidian second-brain vault, mines his Claude Code conversations, tests search, and connects it to his Claude setup ("Morpheus") via MCP. Verdict: works well on quick testing, but he's skeptical of two things — the interactive init (bad for agents) and its custom AI-only compression dialect "AAK."

## Key points (dated)

- 2026-04-07 — Mem Palace was made by Milla Jovovich with her friend Ben; claimed highest-scoring AI memory system ever benchmarked, free, ~5,000 stars. Its structure alone is claimed to improve retrieval by 34%.
- 2026-04-07 — Install/usage flow: `pip install mem palace` → `mem palace init` (interactive, must point at a directory) → mine sources → `mem palace search`. Chuck points it at his Obsidian second brain and mines Claude Code conversations.
- 2026-04-07 — Connects Mem Palace to his Claude assistant "Morpheus" via MCP; on-the-fly test works well and fast, but he notes he already runs a good memory system.
- 2026-04-07 — Two criticisms: the init is interactive ("Great for humans, bad for AI"), and Mem Palace uses a custom AI-only language "AAK" ("Ack") that compresses months of conversations into ~120 tokens — which he distrusts.
