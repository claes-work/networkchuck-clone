## Key claims (dated ‚Äî 2022-05-06)

- `$RANDOM` is a Bash built-in variable that returns a pseudo-random integer between 0
  and 32,767; a different value each time it's referenced, and you never have to set it.
- Linux ships many built-in/system variables beyond `$RANDOM`: `$SHELL` (current shell,
  e.g. Bash), `$USER` (current user account), `$PWD` (print working directory / current
  location), `$HOSTNAME` (the machine's local hostname). All are usable inside scripts.
- You can define your own variable at the shell (`twitter="Elon Musk"`), but a plain
  shell variable is NOT inherited by child processes ‚Äî and scripts are child processes ‚Äî
  so it comes out empty when the script tries to use it.
- `export twitter` promotes a shell variable to an environment variable, which child
  processes (scripts) can then read.
- An exported variable is still lost on logout/reboot; to make it permanent, add the
  `export twitter="Elon Musk"` line to `~/.bashrc`, a script that runs on every login.
- In Linux, a filename beginning with `.` (e.g. `.bashrc`) is hidden; `ls` won't show
  it but `ls -al` (or `ls -a`) reveals it. `.bashrc` lives in the home directory.
- Arithmetic in Bash uses an arithmetic expression: `$(( expression ))` ‚Äî a `$` followed
  by double parentheses with the math inside. Supports `+`, `-`, `*` (asterisk), `/`
  (forward slash), and `%` (modulo).
- Bash does integer math only by default ‚Äî no floating-point/decimals ‚Äî so `2 / 3`
  yields `0` and `10 / 3` yields `3` (the remainder is discarded).
- Modulo (`%`) returns the remainder of a division, e.g. `10 % 3` = 1.
- `$(( RANDOM % 10 ))` bounds the random output to 0‚Äì9 (10 values, counting from zero);
  the pattern `RANDOM % N` yields 0 through N-1. `RANDOM % 15` gives 0‚Äì14; `RANDOM % 20`
  gives 0‚Äì19.
- Project pattern: `getrich=$(( (RANDOM % 15) + age ))` ‚Äî the `RANDOM % 15` must be
  wrapped in its own inner parentheses to isolate it before adding `$age`; the whole
  expression is enclosed in the outer `$(( ))`.
- Practical script-building touches: `sleep 2` to pause, and an `echo "calculating..."`
  loading screen to build suspense/UX flavor.
- Workflow reminders: `nano getrichquick.sh` to edit; shebang line at top; `chmod +x`
  to make executable; verify with `ls -l` (look for the `x` bit); read input with
  `echo`-prompt then a `read`-into-variable step; run with `./getrichquick.sh`.
- Lab setup: Linode spins up an Ubuntu Linux VM in the cloud in seconds for ~$0.01/hour,
  deletable from the context menu when done; new users get $100 credit for 60 days.
