# What is a VMware Engineer? | VMware Certified Professional - VCP | MCSA | CCNA

## Key claims (dated)

All claims 2018-12-22, from the interview. Father-attributed claims are noted; the rest
are Chuck Keith (SUBJECT).

- The father (guest) has been working in VMware for roughly seven years and holds the
  VCP (VMware Certified Professional) certification. (father)
- His first VCP was VCP4; he has recertified across VMware versions (4.5, 5.5, 6, 6.5),
  making him a "VMware history book." (father)
- VMware originally had no certification expiration; like other popular certs, VMware
  later added a two-year expiration requiring recertification. (father)
- You cannot simply self-study and schedule the VMware exam at Pearson the way you can
  with Cisco — VMware requires taking an official VMware class, which can cost upwards
  of $5,000. (both)
- As long as you keep your certification current you do not have to retake that
  (paid) class. (father)
- VMware virtualizes physical equipment/operating systems: a data center that used to
  need ~100 physical servers can be consolidated onto ~15–20 powerful servers running
  many VMs, reducing footprint and making the data center more efficient to run and
  manage. (father)
- Historically each physical server ran one OS doing one thing; now one server can run
  ~20 virtualized servers. (SUBJECT)
- VMware Workstation is a paid option to virtualize your own laptop/desktop and run
  multiple operating systems on it; ESXi is the hypervisor (the kernel for
  virtualization) and can be obtained with a free license (losing some management
  features). (both)
- ESXi can be installed two ways: bare-metal (on the hardware directly) or hosted on
  Windows; bare-metal is the better option because the hosted option consumes host OS
  resources. (father)
- ESXi is Linux-based (Red Hat based). (both, stated in conversation)
- vCenter is introduced to manage all hosts and VMs from a single pane of glass once
  you scale to an enterprise environment. (father)
- VMware features include cloning VMs, built-in disaster recovery, and vMotion, which
  moves a running virtual machine between data centers with no service interruption
  (users don't notice). (both)
- The father's biggest host is a Cisco UCS server with one terabyte of RAM, two
  sockets, and roughly 32 cores per socket; Cisco UCS is itself another certification.
  (father)
- Day-to-day VMware work is largely monitoring: defining alarms in VMware, doing
  morning pre-checks for red alarms, ensuring hosts/hypervisors and VMs run optimally,
  and watching CPU and memory resources so hosts aren't overloaded. (father)
- VMware Hands-on Labs (free on VMware's website) let you touch almost any of their
  technologies in a lab scenario without breaking anything — a recommended learning
  tool. (father)
- The VMware certification track now has roughly three levels: an entry level, an
  administrator level, and VCP (professional), with tracks for data center,
  workstation, networking, and cloud. VCAP (advanced professional) is the next step
  above VCP. (father)
- Recommended prerequisites: basic server knowledge (RAM/CPU resource management). More
  certs are better for a career; a Microsoft MCSA/MCSE combined with a VCP would be an
  "unstoppable" resume. (father)
- When hiring for VMware roles, the father looks primarily for certifications (people
  often list things they can't back up), starts with a phone screen of basic questions,
  then deeper questions. (father)
- Common job titles: large companies may post "VMware engineer," but most roles are
  titled system administrator / system engineer, with VMware as one part of a mixed
  bag; read the job description to see what's actually required. (father)
- To start: plan for your certification, download the exam blueprint from VMware's site
  (e.g., version 6.7) to see what to study; CBT Nuggets released a 6.7 course by Keith
  Barker and Jacob Moran with hands-on labs built in. (SUBJECT)
- VMware is growing, including in the networking space and in the cloud — VMware has
  close relationships with AWS and Google Cloud, and workloads are increasingly moving
  virtually to the cloud. (SUBJECT)
- NetworkChuck credits CBT Nuggets as how he got his start in IT and notes he now
  teaches for them. (SUBJECT)
