# Dump ChatGPT, use Fabric instead

## Summary

A short pitch for [[wiki/entities/daniel-miessler]]'s **Fabric**, described as an
open-source framework for "augmenting humans with AI." Chuck argues most people use AI
wrong and shows what Fabric can do from the command line: summarize a 4-hour YouTube
video with one command, or analyze the claims in a debate and then write an essay from
that analysis — all without leaving the terminal. The short teases a full long-form
video for setup and hands-on use.

## Key points

- (2024-06-07) Fabric is an open-source framework "for augmenting humans with AI."
- (2024-06-07) Example uses: one-command summarization of a 4-hour YouTube video;
  analyzing debate claims and generating an essay from the analysis.
- (2024-06-07) The workflow stays entirely in the command line.
- (2024-06-07) Positioned as a better default than plain ChatGPT for these tasks.
