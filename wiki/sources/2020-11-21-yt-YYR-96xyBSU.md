# i LOVE this switch!! // Cisco Enterprise Switch for SMALL business (Catalyst 1000 series)

## Summary

A sponsored review (paid by Cisco) of the Cisco Catalyst 1000 series switch, framed around a single idea Chuck finds genuinely exciting: an enterprise-grade switch running full Cisco IOS, priced and packaged so small-to-medium businesses can actually deploy it. Chuck's angle as a network engineer is that he has historically had to steer small businesses away from Cisco on cost/complexity — and the Catalyst 1000 removes that objection.

He demos the out-of-box setup (zero-day provisioning via a hold-button "easy mode" + a browser GUI day-zero wizard), praises the fanless/silent design and reliability, and walks through key features: single-IP stacking of up to 8 switches, PoE with perpetual PoE and the "dying gasp" SNMP alert, GUI-or-CLI management, and Bluetooth-adapter management. He then zooms out to the wider Cisco small-business ecosystem (Aironet 1815 AP + wireless controller, Catalyst Wi-Fi 6 AP, Cisco 900 series ISR router). Closing pitch: it's enterprise-grade but affordable, secure, easy to configure, and grows with the business so you avoid a "forklift upgrade" later. He plugs his free CCNA course, which features these switches.

## Key claims

- (2020-11-21) The Catalyst 1000 is an enterprise-grade switch — the kind large businesses deploy — but Cisco is specifically targeting small-to-medium businesses with it; that pairing is unusual and, to Chuck, the whole point. [paraphrase]
- (2020-11-21) It's affordable: Chuck cites the switch available on Amazon "right now for 340" (USD). [paraphrase]
- (2020-11-21) Audience/who it's for: small businesses, SMBs managed by an MSP (managed service provider), and the solo operator ("one guy") keeping his own network up — and also engineers who love the Cisco CLI. [paraphrase]
- (2020-11-21) Setup is easy enough for non-engineers: zero-day provisioning — power on, hold the button ~5 seconds to enter "easy mode," connect port 1 to a computer, browse to 10.0.0.1, log in cisco/cisco, run the day-zero GUI wizard — no command line required. [paraphrase]
- (2020-11-21) It runs Cisco IOS, which Chuck calls the biggest selling point — the same feature-rich OS on Cisco switches/routers for years, configurable via a GUI or the CLI (SSH/console) for those who want it. [paraphrase]
- (2020-11-21) The GUI can monitor ports/connected clients and configure advanced features: static routing, security/ACLs, SPAN (port mirroring), routing protocols, and QoS. [paraphrase]
- (2020-11-21) Reliability is Chuck's core "why Cisco" argument — devices routinely show uptimes of years; "they just work." [paraphrase]
- (2020-11-21) The small model is fanless and silent — quiet enough to sit right next to his microphone. [paraphrase]
- (2020-11-21) You can manage the switch over Bluetooth by plugging a USB Bluetooth adapter into the switch's USB interface and pairing it to a computer. [paraphrase]
- (2020-11-21) Stacking: using the SFP module you can connect up to 8 switches into one logical switch managed from a single IP/portal; an added switch auto-inherits config from the first without manual configuration. He demos adding a 48-port Catalyst 1000. [paraphrase]
- (2020-11-21) PoE powers IP phones, security cameras, and IoT devices; the dashboard shows PoE used vs. unused. [paraphrase]
- (2020-11-21) Perpetual PoE keeps powered devices up through a switch reboot (demoed by rebooting with config saved). [paraphrase]
- (2020-11-21) "Dying gasp": on power loss the switch sends a final SNMP trap so a monitoring solution learns it went down, saving troubleshooting time. [paraphrase]
- (2020-11-21) It scales with the business — an SMB can keep adding switches/gear rather than doing a "forklift upgrade" (rip-and-replace) as it grows into an enterprise. [paraphrase]
- (2020-11-21) Wider ecosystem shown: Cisco Aironet 1815 AP (PoE-powered, runs as an AP and a wireless controller), a newer Cisco Catalyst Wi-Fi 6 AP, and the Cisco 900 series ISR router (VPN, firewall, application visibility/control). [paraphrase]
- (2020-11-21) Chuck offers a free Cisco CCNA course (linked) that teaches CLI configuration and features these switches. [paraphrase]
