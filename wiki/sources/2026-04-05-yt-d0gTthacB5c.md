# Gemma 4 on the iPhone (local AI, no internet required)

## Summary

Chuck demonstrates running Google's new open-source Gemma 4 models fully locally on
an iPhone via Google's AI Edge Gallery app, with the phone in airplane mode (no
internet, no cellular). He downloads the models beforehand, then shows fast text
inference, image understanding (identifying an object in Japan), audio transcription,
and a separate "mobile actions" model that can control device functions like the
flashlight. His takeaway: edge AI computing is now practically useful, not a gimmick.

## Key points

- (2026-04-05) Google released **Gemma 4**, its new open-source AI models; smaller
  variants (the "effective 2 billion" and "effective 4 billion" models) are sized to
  run on a phone.
- (2026-04-05) Everything runs **locally on-device** — no internet, no server calls;
  all inference/compute happens on the phone, demonstrated in airplane mode.
- (2026-04-05) The app used is **Google's AI Edge Gallery**; models must be downloaded
  before going offline.
- (2026-04-05) Capabilities shown: fast text Q&A, image understanding (identifying an
  object photographed in Japan), and audio transcription.
- (2026-04-05) A separate **mobile actions model** (not part of Gemma 4) can trigger
  device actions like turning the flashlight on/off.
- (2026-04-05) Demoed on an iPhone (latest model); Chuck is unsure of the exact
  hardware requirement to run it.
- (2026-04-05) Practical use case: arriving in a foreign country with no internet or
  cellular but still having a working local AI model. <!-- ★ L3-candidate: recurring enthusiasm for local/self-hosted AI as a beliefs theme -->
